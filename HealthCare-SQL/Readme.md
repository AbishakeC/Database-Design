# Healthcare Database System

## 📌 Project Overview
A relational database design for managing hospital operations including patients, doctors, appointments, billing, prescriptions, and payments.

## 🏗 Technologies Used
- MySQL
- SQL (DDL, DML, DCL, TCL)

## 📊 Features
- Normalized database (3NF)
- One-to-many relationships
- Transaction-safe billing
- Role-based access control
- Stored procedures & triggers
